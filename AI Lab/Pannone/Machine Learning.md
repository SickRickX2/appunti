The machine learns how to solve a specific problem alone.
It learns through tunable parameters, machine learning can be categorized in two main types:
- **supervised learning** -> there is someone "helping" the model during the training. Together with the dataset we also provide the solution to the problem.
- **unsupervised learning**
>[!note] Inference
>Second phase of the model.
>We provide only the problem and we expect the correct solution
